Escopo do desafio
Teste Técnico — Microserviço de Comunicação com Filas

Contexto
Hoje possuímos diversos sistemas internos realizando comunicações por canais diferentes, como:
SMS
E-mail
Push notification

Atualmente cada sistema implementa isso de forma isolada, sem padronização e sem rastreabilidade centralizada.

Objetivo
Desenvolver um microserviço responsável por centralizar essas comunicações utilizando filas do Laravel.

Receber solicitações de comunicação;
Registrar a solicitação no banco;
Enviar o processamento para fila utilizando Laravel;
Processar o envio de forma assíncrona;
Registrar logs do processamento e envio.

Requisitos Técnicos
Laravel 10+
Laravel Queue
Docker Compose
Banco relacional (MySQL ou PostgreSQL)
Migrations
API REST
README de execução
Uso de boas práticas e organização de código

Requisitos Funcionais

Endpoint de envio
Criar endpoint: POST /api/communications

Payload esperado
{
  "recipient": "usuario@email.com",
  "channel": "email",
  "subject": "Bem-vindo",
  "message": "Sua conta foi criada com sucesso.",
  "origin_system": "sistema-financeiro"
}


Campos obrigatórios
recipient: Sim
channel: Sim
subject: Não
message: Sim
origin_system: Sim

Canais permitidos
email
sms
push
